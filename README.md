## 📜 项目简介

七律空间是一个展示原创七言律诗的个人网站，融合现代 Web 技术与古典诗词美学：

- 🎨 **视觉设计**：中国传统美学与现代 UI 设计结合
- 🔬 **科学装饰**：晶体结构（SC/BCC/HCP/FCC）与位错滑移面可视化
- 🎵 **听觉体验**：古典音乐背景播放
- 📱 **响应式布局**：完美适配手机、平板、桌面
